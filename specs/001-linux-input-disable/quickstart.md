# 快速开始（计划）

> 说明: 本功能尚未实现，以下为预期的使用与验证流程，用于指导实现与验收。

## 前置条件

- Linux 环境
- 具备访问输入设备节点的权限（建议通过用户组授权）

## 预期使用流程

1. 安装并启动服务
2. 使用 CLI 开启功能: `aegis-input enable`
3. 插入外置键盘，验证内置键盘自动禁用
4. 移除外置键盘，验证内置键盘自动恢复
5. 插入外置鼠标，验证内置指点设备自动禁用
6. 移除外置鼠标，验证内置指点设备自动恢复
7. 关闭功能: `aegis-input disable`，验证内置设备保持可用
8. 查询状态: `aegis-input status`

## 配置文件

- 默认路径: `~/.config/aegis-input/config.toml`
- 可通过环境变量 `AEGIS_INPUT_CONFIG` 指定路径

## 预期验收要点

- 外置设备接入/移除在 2 秒内触发对应内置设备的禁用/恢复
- 外置键盘仅影响内置键盘，外置鼠标仅影响内置指点设备
- 多个同类型外置设备存在时，只有全部移除才恢复
- 空闲时资源占用满足规格中的 CPU/内存预算
