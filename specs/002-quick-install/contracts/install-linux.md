# 接口契约: Linux 安装脚本

## 目的

提供一键安装、卸载与状态检查的命令入口。

## 命令

### `install/linux/install.sh`

- **作用**: 安装二进制与 systemd 服务，并启动服务
- **输入**: 无（可选参数: `--prefix`、`--user`）
- **输出**: 安装完成提示与下一步指引
- **退出码**: 0 成功，非 0 失败

### `install/linux/uninstall.sh`

- **作用**: 卸载服务与相关文件
- **输入**: 无
- **输出**: 卸载完成提示
- **退出码**: 0 成功，非 0 失败

## 行为约束

- 安装完成后必须启动并 enable systemd 服务
- 卸载必须停止服务并清理 unit 文件与安装元数据
- 失败必须输出可操作的错误提示
