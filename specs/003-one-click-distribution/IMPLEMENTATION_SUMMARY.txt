使用方法
================================================================================

一键安装:
  curl --proto '=https' --tlsv1.2 -sSf \
    https://raw.githubusercontent.com/Yu-Xiao-Sheng/aegis-input/main/install/remote/install.sh | bash

检查更新:
  aegis-input --check
  或
  curl ... | bash -s -- --version-check

创建发布:
  git tag v0.3.0
  git push origin v0.3.0
